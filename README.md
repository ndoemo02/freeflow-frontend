# FreeFlow Frontend

Aplikacja frontend do FreeFlow - narzędzie do transkrypcji audio z wykorzystaniem sztucznej inteligencji.

## Technologie
- React + Vite
- Tailwind CSS
- JavaScript

## Instalacja

1. Sklonuj repozytorium:
```bash
git clone https://github.com/ndoemo02/freeflow-frontend.git
cd freeflow-frontend
```

2. Zainstaluj zależności:
```bash
npm install
```

3. Uruchom aplikację:
```bash
npm run dev
```

## Struktura
- `/assets` - pliki CSS, JS i obrazy
- `/api` - konfiguracja API
- `/dist` - zbudowana aplikacja

## Wdrożenie
Aplikacja jest skonfigurowana do automatycznego wdrożenia na różnych platformach.
